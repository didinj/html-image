HTML Image

Put an Image on a Web Page

Syntax:
<img src="twitter.png">

Alt Attribute

Use the "alt" attribute to provide an alternate text for an image. This attribute will show text if your connection to slow or, if a browser cannot find an image.

Syntax:
<img src="facebook.png" alt="Facebook">

Width and Height or Style Attribute

Width and Height Attribute

Syntax:
<img src="flower.jpg" alt="Flower" width="220px" height="140px">

Style Attribute

Syntax:
<img src="beach.jpg" alt="Beach" style="width:220px; height:140px">

Images in Another Folder or Server

In Another Folder

Syntax:
<img src="img/smartphone.jpg" alt="Smartphone" width="200px" height="100px">

In Another Server

Syntax:
<img src="https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png" alt="Google">

Animated Images

Syntax:
<img src="google.gif" alt="Google" width="400px" height="250px">

Image as a Link

Syntax:
<a href="https://google.com">
<img src="google.gif" alt="Google" width="400px" height="250px">
</a>

Image Floating

Syntax:
<p> <img src="Smiley.svg" alt="Google" style="float:left;width: 30px; height: 30px"> </p>
